# Calculadora de Áreas (Tkinter)

Aplicação com interface gráfica para calcular a área de seis figuras geométricas:
triângulo, quadrado, retângulo, círculo, losango e trapézio.

## Funcionalidades

- Interface gráfica simples com um botão para cada figura
- Entrada de dados via caixas de diálogo
- Cálculo automático e exibição do resultado

## Como executar

Requer Python 3 com Tkinter (já vem incluso na instalação padrão do Python).

```bash
python areas.py
```

## Tecnologias

Python 3, Tkinter, módulo math
